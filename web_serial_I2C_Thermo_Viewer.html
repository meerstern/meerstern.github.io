AMG88XX Thermo Array Viewer


USBSerial + SC18IM704+ AMG88XX

Note:
This tool is a tool for testing AMG88XX Thermo Array Sensor using USB Serial I2C Converter Board and a browser.
The USB Serial I2C Converter Board V1 equipped with SC18IM700 is not supported due to buffer size limitations.
All USB Serial I2C Converter Boards currently on sale are V2.

Works offline and locally

Only Edge or Chrome browsers are supported
